
Depending on the language you should use a suite of code quality tools:

* Bash: [[./languages/bash-build]]
* Java: [[./languages/java-build]]
* Python: [[./languages/python-build]]
* R: [[./languages/r-build]]
* SQL: [[./languages/sql-build]]
* Typescript: [[./languages/typescript-build]]
* Polyglot: [[./languages/polyglot-build]]

Other languages will need to be independently researched. For a emergent design project you will need tools to support the following functions:

1. Syntax & Formatting Validation
2. Strict Type-Checking
3. Testing & Assertion
4. Logging
5. Dependency & Vulnerability Scans
6. Code coverage
7. Documentation consistency
8. Code duplication detection
9. Profiling
