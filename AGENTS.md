# Agent skills

This is a repository of agent skills.

It is organised by bundles in top level sub directory.

When creating new skills in this project make sure you place them in a bundle sub directory.

Skills in the `ai-training-bundle/` are tools to use now. They should be symlinked to your skills directory when they are complete.

Whe you create a new skill in the `ai-training-bundle/` directory offer to symlink to it from `.github/skills`, `.claude/skills`, and `.opencode/skills`
